SEPIA Speech-To-Text Server

The SEPIA client app can handle speech recognition in two different ways, either using platform specific engines (available in Android, Chrome, Edge, Safari, etc.) or using your own, independent, offline speech recognition server that can be configured to use customized acoustic and language models. To learn how to set up and use the server please visit:
